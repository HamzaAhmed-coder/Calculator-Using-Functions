# Calculator Using Functions

## Description
This project is a simple calculator developed in C++ using functions. It performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division through a menu-driven program.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu

## Concepts Used
- Functions
- Conditional Statements
- Loops
- User Input/Output

## Language
C++
